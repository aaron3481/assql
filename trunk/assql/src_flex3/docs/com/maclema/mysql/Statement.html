executeQuery()method 
public function executeQuery(sqlString:String = null):MySqlToken

Executes the specified sql statement. The statement can be provided using either the sql property or as the first parameter of this method. You may also specify a IResponder object as the second parameter.

When result(data:Object) is called on the IResponder the data object will be either a ResultSet, in the case of query statements, and in the case of data manipulation statements, will be an object with two properties, affectedRows, and insertID.
